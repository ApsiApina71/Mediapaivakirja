# Mediapaivakirja

* Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
* Käyttäjä pystyy lisäämään teoksia tietokantaan sekä muokkaamaan ja poistamaan omia lisäyksiään.
* Käyttäjä näkee kaikki sovellukseen lisätyt teokset.
* Käyttäjä pystyy etsimään teoksia hakusanalla.
* Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja (esim. kirjoitettujen arvostelujen määrän) sekä listan käyttäjän lisäämistä teoksista.
* Käyttäjä pystyy valitsemaan teokselle yhden tai useamman luokittelun (esim. komedia, kauhu, tietokirjallisuus).
* Käyttäjä pystyy antamaan teokselle arvostelun (tähtiarvio 1-5 ja tekstimuotoinen kommentti). Teoksen sivulla näytetään kaikkien käyttäjien jättämät arvostelut ja arvosanojen keskiarvo.
